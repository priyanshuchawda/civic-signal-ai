# CivicSignal AI

Delhi-first AI decision intelligence for safer, healthier, and more responsive communities.

## Local Setup

```bash
npm install
npm run dev
```

Create `.env.local` from `.env.example` when enabling Gemini:

```bash
GEMINI_API_KEY=your_key_here
GEMINI_MODEL=gemini-2.5-flash
```

No paid infrastructure or GitHub Actions are required for the current phase.
